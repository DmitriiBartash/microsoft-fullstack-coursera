> Practice turning a plain-language **algorithm** into working **C#**. You start from a
> problem statement, sketch the logic as **pseudocode**, then translate that pseudocode

## 🎯 Objective

Learn to move from a **problem statement** to **pseudocode** to **runnable C#**, so that the
control flow you describe in plain language (loops, conditionals, accumulators) maps cleanly
onto real language constructs like `foreach`, `if`, and compound-assignment operators.

## 🧩 Walkthrough

### Problem 1 — Calculating the sum of an array

**Problem statement:** Write an algorithm that calculates the sum of all numbers in a given
array of integers.

**1. Pseudocode**

```text
Start
  Create variable sum and set it to 0
  For each number in the array
    Add number to sum
  End For
  Display the value of variable sum to user
End
```

**2. Translate to C#**

```csharp
// Initialize an array of integers
int[] numbers = { 5, 8, 3, 4, 2 };

// Initialize the sum variable
int sum = 0;

// Loop through each number in the array
foreach (int number in numbers)
{
    sum += number;
}

// Output the sum of the array
Console.WriteLine("The sum of the array is: " + sum);
```

Each line of pseudocode has a direct C# counterpart: *"Create variable sum and set it to 0"*
becomes `int sum = 0;`, *"For each number in the array"* becomes `foreach (int number in numbers)`,
and *"Add number to sum"* becomes `sum += number;`. With `{ 5, 8, 3, 4, 2 }` this prints
`The sum of the array is: 22`.

### Problem 2 — Counting the number of vowels in a string

**Problem statement:** Write an algorithm to count the number of vowels (`a, e, i, o, u`) in a
given string.

**1. Pseudocode**

```text
Start
  Create variable vowelCount and set to 0
  For each character in the string
    If character is a vowel
      Add 1 to vowelCount
  End For
  Print vowelCount
End
```

**2. Translate to C#**

```csharp
int vowelCount1 = 0;
char[] letters = ['h', 'e', 'l', 'l', 'o'];
char[] vowels = ['a', 'e', 'i', 'o', 'u'];

foreach (char letter in letters)
{
    if (vowels.Contains(char.ToLower(letter)))
    {
        vowelCount1 += 1;
    }
}

Console.WriteLine(vowelCount1);
```

The *"If character is a vowel"* test is implemented with `vowels.Contains(...)`. Calling
`char.ToLower(letter)` first makes the check **case-insensitive**, so an uppercase `'E'` still
counts. For the characters `['h', 'e', 'l', 'l', 'o']` this prints `1`.

> The collection-expression syntax `['a', 'e', ...]` requires **C# 12 / .NET 8** or later. On an
> older target, use the classic form `new char[] { 'a', 'e', 'i', 'o', 'u' }` instead.

## 🔑 Key concepts

- **Pseudocode bridges idea and code** — describe the *logic* in plain language first, then each
  line translates almost one-to-one into a C# statement.
- **The accumulator pattern** — initialize a variable (`sum = 0`, `vowelCount = 0`), update it
  inside a loop, and read it after the loop; it underlies counting, summing, and many aggregations.
- **`foreach` for iteration** — when you need every element but not its index, `foreach` is the
  clearest way to express *"for each X in the collection."*
- **Membership tests + normalization** — `vowels.Contains(char.ToLower(letter))` shows how a
  conditional becomes a lookup, and why normalizing case first makes the comparison robust.
